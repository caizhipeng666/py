# 集合

### 查找最大/最小的N个元素

```
import heapq

num = [1, -2, 2, 0.1,  9, 0]

print(heapq.nlargest(2, num))
print(heapq.nsmallest(3, num))

test = [
    {"name": "czp", "money": 100},
    {"name": "cpp", "money": 110},
    {"name": "c2p", "money": 90},
    {"name": "Czp", "money": 100},
]

small = heapq.nsmallest(1, test, key=lambda p: p["money"])
print(small)
```
> 适合筛选较少个元素时使用
