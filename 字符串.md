### bytes & bytesarray
> 各个元素是介于(0~255)的整数

```python
str编码之后 -> bytes

如果不指定字符集，默认使用 UTF-8 字符集
a = "test"
b = a.encode("utf-8")

调用 bytes()函数(其实是 bytes 的构造方法)将字符串按指定字符集转换成字节串，
如果不指定字符集，默认使用 UTF-8 字符集
c = bytes('test',encoding='utf-8')

如果字符串内容都是 ASCII 字符，则可以通过直接在字符串之前添加b来构建字节串值
d = b"test"
```
