# 字典

### 合并映射
```python
from collections import ChainMap

a = {"x":1, "z":3}
b = {"y":2, "z":4}

ChainMap(a, b)

>>> ChainMap({'x': 1, 'z': 3}, {'y': 2, 'z': 4})
```

### 只读字典
```python
from types import MappingProxyType

data = {'a' : 'b'}
test = MappingProxyType(data)
test["b"] = 1

>>> TypeError: 'mappingproxy' object does not support item assignment
```
